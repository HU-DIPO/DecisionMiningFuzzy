Decision mining

Welcome,

Using the tools on this webapp you can analyse your decision processes. We automatically infer the rules your decision model uses based on the data you upload.

The tools work by analysing your data. Your data must be a .csv-file, with its values delimited with commas (“,”). The first row should contain the column names. Each of these columns can be marked as input or output, and continuous or categorical. If you do not wish to use one of the columns in your dataset, simply press “exclude”.
