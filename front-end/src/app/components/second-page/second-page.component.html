Decision mining

Input/Output, Continue/Categoriaal:
A decision process may only have one output, yet many inputs. Columns can be marked as input, output, or be excluded all together. Furthermore, columns may be marked as continuous or categorical. If your column contains categories (e.g., [1, 2, 3], or [“sunny”, “windy”, “rainy”]), it is categorical. If it contains numbers in a certain range (e.g., [1, 2, 3], or [0.5, 100, 13.4]), it is continuous.
